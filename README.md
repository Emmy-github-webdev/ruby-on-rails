# Ruby on Rails

> Learn Ruby on Rails from scratch.


## Built With

- Ruby

## Getting Started

- You will need to have Ruby installed.

- Clone this repo [Ruby on Rails](https://github.com/Emmy-github-webdev/ruby-on-rails).

## Run Ruby on Rails

- cd the directory
- Run rails server
